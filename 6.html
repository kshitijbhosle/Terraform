Exercise 4 - Using Terraform to create databases.

Prerequisite : Mysql or Mariadb must be configured before.

provider "mysql" {

  endpoint = "172.17.0.51:3306"

  username =  Mysql_User

  password =  Mysql_Password

}

# Create a Database

resource "mysql_database" "app" {

  name = "my_awesome_app"

}

 

Use terraform init command to initialise the provider.

Check the changes using terraform plan command.

Terraform plan

To apply those changes use terraform apply command.

terraform apply

To check the changes

mysql -uroot -p

MariaDB [(none)]> show databases;
