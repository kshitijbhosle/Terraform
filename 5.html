Exercise 3 - Creating containers using Terraform

Terraform File:

 terraform {

  required_providers {

    docker = {

      source = "terraform-providers/docker"

    }

  }

  required_version = ">= 0.13"

}

resource "docker_image" "nginx" {

  name = "nginx:latest"

}

resource "docker_container" "nginx" {

  image = docker_image.nginx.latest

  name  = "tutorial"

  ports {

    internal = 80

    external = 80

  }

}

Use terraform init command to initialise the provider.

Check the changes using terraform plan command.

Terraform plan

To apply those changes use terraform apply command.

terraform apply
