Exercise 2 - Create an EC2 instance using Terraform

Get access_key and secret_key from aws dashboard.

provider "aws" {

  region          = "us-east-1"

  access_key =          

  secret_key  =

  token           =

}

resource "aws_instance" "web" {

  ami                  = "ami-0be2609ba883822ec"

  instance_type = "t2.micro"

}

Use terraform init command to initialise the provider.

Check the changes using terraform plan command.

Terraform plan

To apply those changes use terraform apply command.

terraform apply

Check the EC2 dashboard to see our newly created instance.
